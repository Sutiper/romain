Configuration de Base

← Retour Synthèse

Cette page présente une synthèse des principales commandes de configuration de base sur les équipements réseau Cisco (switch et routeur), incluant la modification du hostname, la configuration des mots de passe, la sauvegarde et la réinitialisation de la configuration, la création et l'attribution de VLAN, la configuration des adresses IP, la mise en place d'un trunk, l'utilisation des sous-interfaces pour le routage inter-VLAN, la gestion du protocole STP ainsi que l'activation d'un accès distant via Telnet.

Modification du hostname (switch/router)

(config)#hostname 'nom'

Mot de Passe Enable (switch/router)

(config)#enable password 'mot de passe'

Mot de Passe Enable sécurisé (switch/router)

(config)#enable secret 'mot de passe'

Mot de Passe de Console (switch/router)

(config)#line console 0
(config-line)#password 'mot de passe'
(config-line)#login

Mise en place d'un VLAN (switch)

(config)#vlan 'num du vlan'
(config-vlan)#name 'nom du vlan'
(config-vlan)#exit
(config)#interface fastethernet 'num port'
(config-if)#switchport mode access
(config-if)#switchport access vlan 'num du vlan'

Configuration IP (switch)

(config)#interface vlan 'num du vlan'
(config-if)#ip address 'ip du switch' 'masque'
(config-if)#no shutdown
(config)#ip default-gateway 'ip du routeur'

Configuration trunk (switch)

(config)#interface fastethernet 'num du port'
(config-if)#switchport mode trunk
(config-if)#switchport trunk allowed vlan 1,10,20,30 (optionnel)

Configuration IP (router)

(config)#interface GigabitEthernet 'num interface'
(config-if)#ip address 'ip du routeur' 'masque'
(config-if)#no shutdown

Configuration ip sous interface (router)

(config)#interface fastethernet 'num interface'.'num sous-interface'
(config-subif)#encapsulation dot1Q 'num vlan'
(config-subif)#ip address 'ip' 'masque'

Activation/Désactivation STP (switch)

Désactiver

(config)#no spanning-tree vlan 'num vlan'

Activer

(config)#spanning-tree vlan 'num vlan'

Priorité STP (switch)

(config)#spanning-tree vlan 'num vlan' priority 'valeur'

Telnet (switch/router)

(config)#line vty 0 4
(config-line)#password 'mot de passe'
(config-line)#login
(config-line)#transport input telnet

Sauvegarde (switch/router)

#copy running-config startup-config

Reset (switch/router)

#erase startup-config
#reload
